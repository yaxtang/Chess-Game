# Chess-Game
A two player Chess Game.

## Environment Setting
- IDE: IntelliJ
- Language: Java, HTML
- Tools: JFrame, Swing, GUI

## Features
- Add two new pieces Elephant and Spy with different moving methods
- REDO function
- Detections for possible path of current piece.
